# 🏥 Hospital Management System (HMS)

## 📌 Introduction

Hospital Management Systems are widely used across the globe to streamline and digitize hospital operations. This application provides an intuitive desktop interface to manage doctors and patients efficiently, backed by a reliable MySQL database.

---

## 🛠️ Technology Stack

| Layer | Technology |
|---|---|
| Programming Language | Java (High-Level) |
| GUI Framework | Java Swing |
| Database | MySQL (DBMS) |

---

## 🗂️ Modules

### 1. 🔐 Login Page
Secure authentication to control access to the system.

---

### 2. 👨‍⚕️ Doctors Module

Manage the hospital's medical staff with the following operations:

| Feature | Description |
|---|---|
| **Add Doctor** | Register a new doctor with ID, Name, and Specialization |
| **Edit Doctor** | Update a doctor's Name and Specialization |
| **Fire Doctor** | Remove a doctor from the system by Doctor ID |
| **View Doctors** | Browse all registered doctors and their details |

---

### 3. 🧑‍⚕️ Patient's Module

Handle patient admissions and records with the following operations:

| Feature | Description |
|---|---|
| **Admit Patient** | Register a patient with ID, Name, Disease, Admit Date & Time |
| **Edit Patient** | Update patient's Name, Disease, Admit Date and Time |
| **Discharge Patient** | Discharge a patient from the system by Patient ID |
| **View Patients** | Browse all patient records and details |
